# PopCornFlix

# About
iOS movie application based on VIPER architecth

# Requirements
* Xcode 12 or later
* iOS 14 or later

# Getting Started
* Clone or download
* Pod install
* Build and Run

# Technical Features
* VIPER Arhitecture
* Using UIKit with programmatically

# Features
* Discover popular movies
* Search movies according to title
* Add favorite movies to bookmark
